## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/fvdime/art-exhibition.git
    ```
   
2. Install NPM packages:
   ```bash
   npm install
    ``` 
   
3. Add .env file:
   ```bash
   DATABASE_URL="<DATABASE URL>"
   NEXT_PUBLIC_AMAZON_S3_BASE_URL="<AWS S3 BASE URL>"
    ```

4. Prisma Commands:

   ```bash
   npx prisma db push
   npx prisma generate
   ```   

5. Run the development server:

   ```bash
   npm run dev
   ```
